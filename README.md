# ESP32-DailyExpenseTracker

still manual, i built this because i sometimes dont know how i manage money 
fun project for me to learn c++, and esp32's

## Things i use 

- a microcontroller (esp32-32D)
- 3 push buttons
- 1 OLED 0.96 128x64

## Diagram
| ESP32  | OLED |
| --- | --- |
| 21 | SDA  |
| 22 | SCL  |
| 3v3 | VCC |
| GND | GND  |

### 16,17,18 pin are buttons
